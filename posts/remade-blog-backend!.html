So yesterday and today, I decided to remake the backed for this crappy blog.
Initially I made it so that there was a single file that would contain every single post.

It worked, but in practice, it was terrible. Everytime you loaded the home page or a blog page you were downloading the entire website worth of posts...
Now the site has a /posts folder and a database json file that keeps track of all the topics for posts, as well as just the titles and dates of them.

I was able to make the site dynamically generate a new page via the GitHub API. It's kinda fun to mess around with.
Although there really is no benefit to doing this for a blog, it's been fun and I think well worth it because I got to toy with a silly concept.
